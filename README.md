# Bank-Loan-Analysis-Excel-Dashboard-
This project is an Excel-based data analysis dashboard designed to analyze bank loan data and generate meaningful insights. It helps in understanding loan distribution, customer behavior, and financial trends using interactive visuals.
This Excel file contains 4 sheets, each serving a different purpose in the analysis process.

This project is created to showcase data analysis skills using Excel, which is useful for roles like:
Data Analyst
Business Analyst
Finance Analyst

1. Raw Data
Contains original loan dataset
Includes customer details, loan amount, status, etc.
Used as the base for analysis

2. Data Cleaning / Processing
Data is cleaned and structured
Removed duplicates, handled missing values
Prepared for analysis

3. Analysis / Pivot Tables
Pivot tables used for summarizing data
Key metrics like:
    -Total Loan Amount
    -Loan Status Distribution
    -Customer Segmentation

4. Dashboard
•   Interactive dashboard with charts and visuals
•   Easy-to-understand insights
•   Helps in quick decision-making

Tools & Skills Used
•	Microsoft Excel: Primary tool for data modelling, visualization, and interactivity.
•	Excel Charts: Column, bar, donut, line, and map charts for data representation.
•	Pivot Tables & Slicers: For interactive filtering and summarizing large datasets.
•	Conditional Formatting: Highlighting trends and performance indicators.
•	Formulas & Functions: Including SUMIFS, VLOOKUP, IF, and TEXT.

1. Summary Dashboard
•	Focus: High-level metrics and performance snapshot.
•	Key Components:
   o	Total Loan Applications, Funded Amount, Amount Received
   o	Average Interest Rate and DTI (Debt-to-Income Ratio)
   o	Good vs Bad Loan Issuance (Donut Charts)
   o	Status-wise Loan Breakdown (Active, Paid Off, Defaulted, etc.)
2. Overview Dashboard
•	Focus: Trend and demographic analysis.
•	Key Components:
   o	Loan Application Trends by Month
   o	Geographic Distribution by U.S. States (Map Chart)
   o	Application Count by Loan Term and Purpose



Key Outcomes
•	Delivered a visually-rich and interactive Excel dashboard accessible to non-technical stakeholders.
•	Identified 50% split between good and bad loans, enabling targeted risk management.
•	Tracked monthly performance trends (MoM/MTD) to monitor loan disbursal growth.
•	Enabled data-driven strategy by revealing patterns in customer demographics and regional loan behaviour.
•	Provided a single-view solution to assess financial health, risk exposure, and growth opportunities in the loan portfolio.
