# Bank-Loan-Analysis-Excel-Dashboard-
This project is an Excel-based data analysis dashboard designed to analyze bank loan data and generate meaningful insights. It helps in understanding loan distribution, customer behavior, and financial trends using interactive visuals.
This Excel file contains 4 sheets, each serving a different purpose in the analysis process.

1. Raw Data
Contains original loan dataset
Includes customer details, loan amount, status, etc.
Used as the base for analysis

3. Data Cleaning / Processing
Data is cleaned and structured
Removed duplicates, handled missing values
Prepared for analysis

5. Analysis / Pivot Tables
Pivot tables used for summarizing data
Key metrics like:
    -Total Loan Amount
    -Loan Status Distribution
    -Customer Segmentation

7. Dashboard
Interactive dashboard with charts and visuals
Easy-to-understand insights
Helps in quick decision-making

Tools & Skills Used
-Microsoft Excel
-Pivot Tables
-Charts & Graphs
-Data Cleaning
-Data Analysis

This project is created to showcase data analysis skills using Excel, which is useful for roles like:
Data Analyst
Business Analyst
Finance Analyst
